# Começando com Angular
## Comandos
- Instalação do angular globalmente
```sh
npm install -g @angular/cli
```

- Cria um protejo Angular
```sh
ng new projectName
```

- Executa a aplicação
```sh
ng serve
```

- Caso o sistema não permita executar o ng serve
```sh
Set-ExecutionPolicy RemoteSigned
```

- Build para produção
```sh
ng build --prod
```
